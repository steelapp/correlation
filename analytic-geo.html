Equation of a Line

Slope-Intercept Form

y = mx + b

Point-Slope Form

y - y₁ = m(x - x₁)

Intercept Form

x/a + y/b = 1
  • Parallel: Equal slopes (m₁ = m₂)
  • Perpendicular: Negative reciprocals (m₁ = -1 / m₂)

Distance Formulas

Point to a Line

d = |Ax₁ + By₁ + C| / √(A² + B²)

Between 2 Parallel Lines

d = |C₂ - C₁| / √(A² + B²)

Angle Between 2 Lines

tan θ = (m₂ - m₁) / (1 + m₁m₂)
  • General Equation of a Line: Ax+By+C=0
  • Standard Form: Ax+By=C
Conic Sections

General Conic Equation

🔄 Click to Flip

Discriminant B² - 4AC

Ax² + Bxy + Cy² + Dx + Ey + F = 0

B² - 4AC = 0 (Parabola)
B² - 4AC < 0 (Ellipse)
B² - 4AC > 0 (Hyperbola)

If B = 0 (No xy term):

  • A = C : Circle
  • A = 0 or C = 0 : Parabola
  • A ≠ C (same sign) : Ellipse
  • A & C (opp. signs) : Hyperbola

Standard & General Forms

Standard Center (h,k)

(x - h)² + (y - k)² = r²

General Form

x² + y² + Ax + By + C = 0

Properties:

  • If center is at the origin (0,0), the formula simplifies to x² + y² = r².
  • To convert from General to Standard, complete the square for both x and y.

Standard Forms & Distances

Up/Down

(x - h)² = ±4a(y - k)

Left/Right

(y - k)² = ±4a(x - h)
  • a = Distance from Vertex to Focus = Distance from Vertex to Directrix.
  • Latus Rectum: L = 4a. It passes through the focus, parallel to the directrix.

Standard Form & Properties

Major Axis Parallel to X

(x - h)² / a² + (y - k)² / b² = 1

Major Axis Parallel to y

(y - k)² / a² + (x - h)² / b² = 1
  • a > b
  • a² = b² + c²
  • LR =2b²/a
  • d=a/e
  • e=c/a

Standard Form & Asymptotes

Transverse Axis Parallel to X

(x - h)² / a² - (y - k)² / b² = 1

Transverse Axis Parallel to y

(y - k)² / a² - (x - h)² / b² = 1
  • a > b or a < b, but +a
  • a² + b² = c²
  • LR =2b²/a
  • d=a/e
  • e=c/a
  • assymptote: y ± (b/a)x
