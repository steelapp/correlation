Quadratic Equation

  • > 0: Roots are real & unequal

Age Problems

🔄 Click to Flip

Age Problems

  • If the current age is x, then n times the age is nx.
  • If the current age is x, then the age n years later = x + n
  • If the current age is x, then age n years ago = x – n
  • The ages in a ratio a:b will be ax and bx
  • If the current age is x, then 1/n of the age is x/n.

x = present, x + n = n years later, x - n = n years ago

Work Problems

🔄 Click to Flip

Work Problems

Case 1: Each worker has different rates
• rate = job done / time
• combined rates = rate 1 ± rate 2

Different rates, staggered:
• 1 = (1/A)x + (1/B)y
  - A and B are their solo total times
  - x and y are the actual hours/days they spent working.

Different rates, synchronous:
• 1/T = 1/t1 + 1/t2

Case 2: Each worker has the same rate
• job done = (number of men)(number of days)
• WT = W1T1 + W2T2 + WnTn
• combined rates = rate 1 ± rate 2

Remainder Theorem

🔄 Click to Flip

Remainder Theorem

  • If g(x) = ax + b (linear), remainder is constant (c)
  • If g(x) = ax2 + bx + c (quadaratic), remainder is ax + b (linear)
  • If g(x) = ax3 + bx2 + cx + d (cubic), remainder is ax2 + bx + c (quadaratic)

Permutation

  • Theorem 2: n objects taken all at a time in which n1 of the objects are alike, n2 are alike, n3 are alike, and so on
